### Cara Menggunakan
Install Depedency
```bash
composer install
```

Install Package
```bash
npm install
```

Copy Env File
```bash
cp .env.example .env
```

Generate Key
```bash
php artisan key:generate
```

Configurasi Database
```php
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db_megah_marmer
DB_USERNAME=root
DB_PASSWORD=
```

Migrate Table
```bash
php artisan migrate
```

Seeder Data
```bash
php artisan migrate:fresh --seed
```

Aktifasu Storage
```bash
php artisan storage:link
```

Running atau Jalankan Project
```bash
npm run start
```

Kunjungi link ini `http://127.0.0.1:8000/`
